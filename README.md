# 🚗 Driver Drowsiness Detection System (Computer Vision)

This project is a **Driver Drowsiness Detection System** built using **Computer Vision and MediaPipe**.
It monitors the driver's eye closure and yawning behavior in real-time to detect fatigue and prevent accidents.

---

## 🚀 Features

* 👁️ Eye closure detection (closed eye count)
* 😴 Yawning detection (yawn count)
* 📸 Real-time monitoring using webcam
* ⚠️ Drowsiness detection and alert system
* 🖥️ Simple GUI interface using Tkinter
* ⚡ Fast and efficient processing

---

## 🛠️ Technologies Used

### Core:

* Python
* OpenCV
* MediaPipe

### Libraries:

* NumPy
* Tkinter
* PIL (Pillow)
